# GhostPilot

GhostPilot is an offline AI-powered Network Copilot developed for ISRO Hackathon PS-13.

## Features

- Offline AI using Ollama
- Phi-3 Language Model
- Network Rule Engine
- AI-based Network Analysis
- JSON Telemetry Input

## Tech Stack

- Python
- Ollama
- Phi-3
- JSON

## Run

```bash
python app.py
```
