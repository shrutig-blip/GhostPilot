GhostPilot Frontend - Phase 1

Project Overview

Build a React + Vite frontend for GhostPilot, an AI-powered Predictive Network Monitoring Dashboard developed for the ISRO Hackathon.

Important: Build only the frontend. Do not generate any backend or server code.

The goal is to create a modern, responsive, production-quality dashboard that resembles an ISRO Mission Control Center for monitoring network telemetry and AI-generated predictions.
